COVID-19 Project

A project to track back how the COVID-19's first case appeared, how the world confronted the pandemic, and the vaccination for coronavirus emerged. I focus on the top 5 countries I'm interested in: Vietnam, China, Germany, Finland, and the United States.

Tableau

Using SQL Server Management Studio (SSMS), I clean the data and create the necessary views and tables.

After all the essential calculations, I connect to Tableau Desktop to visualize the data and create a dashboard. The final dashboard is published to Tableau Public.

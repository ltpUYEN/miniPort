E-commerce Analysis

I used a fictitious dataset of online pet supply company from Datacamp to learn about e-commerce businesses. First, data exploration with the dataset aiming to build sales and expense analysis. Filtering out invalid data, creating metrics and using a self-join to create a correlation matrix then, using it to build market basket analysis. Adding state-region mapping, building lifetime Value, hypothetical what-if scenario, key KPIs, etc.

Creating a cohesive Tableau story with related dashboards. Making sure the analytics pipeline runs smoothly and effectively communicate the target insight. The dashboard showed the product and customer attributes that will result in the highest ROI for the upsell and cross-sell scenarios.

SUPPLY CHAIN ANALYTICS

I used this simulated dataset to learn about the supply chain analyst's general view. I also learned some key calculations and KPIs to apply for this supply chain aspect.

Starting by using a quote analysis tool to create a data model. Then, making a volume parameter and scenario analysis for cost calculation change as volume changes, adding row-level security to label the supplier to access the data depending on their roles. The volume parameter created a more extensive view for scenario volume analysis. The dashboard also showed the comparison between the buy and make options for suppliers.
